# Health Tracker
PCでのゲームやブラウザ利用時間を自動で記録し、長時間使用を防ぐことで健康的な生活を支援する。

## 作品概要

**タイトル：** PC使用時間管理・健康支援アプリ

**目的：**
PCでのゲームやブラウザ利用時間を自動で記録し、
長時間使用を防ぐことで健康的な生活を支援する

### ■ 解決する社会課題
- ゲーム・動画のやりすぎ
- 長時間PC使用による健康悪化
- 自己管理の難しさ

👉 利用時間の可視化＋通知で改善を促す

## ■ 主な機能

### ① 使用時間の自動記録
- 操作中のアプリを取得
- 使用時間を継続的に記録

👉 アクティブウィンドウのみ計測

### ② シンプルな分類
- ゲーム
- ブラウザ（例：Google Chrome）
- その他

### ③ ゲームの詳細表示
例：
- VALORANT
- Apex Legends

👉 ゲーム全体＋内訳を表示

### ④ 通知機能
- 一定時間経過で通知（例：1時間）

👉 長時間利用の防止

### ⑤ 除外機能
例：
- Discord

👉 不要なアプリは記録しない

### ⑥ 多言語対応（日本語・英語）
- Game / ゲーム
- Browser / ブラウザ

👉 表示を切り替え可能

## ■ 判定方法（シンプル設計）
1. 除外アプリ → 無視
2. 既知ゲーム（exe名） → ゲーム
3. `chrome.exe` → ブラウザ
4. それ以外 → その他

👉 ルールベースで実装

## ■ 表示イメージ

### ■ メイン
- ゲーム：3時間
  - VALORANT：2時間
  - Apex：1時間
- ブラウザ：2時間
- その他：1時間

## ■ 動作の流れ
起動 → 1秒ごとにアクティブアプリ取得 → 分類 → 時間加算 → 一定時間で通知 → データ保存

## ■ 画面構成
- 普段 → タスクトレイ常駐
- クリック → 使用時間表示

## ■ 技術構成
- 言語：C#
- 開発：Microsoft Visual Studio
- 形式：Windowsデスクトップアプリ
